# 📰 今日の話題 - Daily Topics

静的ウェブサイトで、毎日の話題を提供するサイトです。スマートフォンに完全対応したレスポンシブデザインになっています。

## 🌟 特徴

- ✅ 完全レスポンシブ対応（スマートフォン・タブレット・デスクトップ）
- ✅ カテゴリ別フィルタリング機能
- ✅ モダンなUIデザイン
- ✅ 高速ローディング（静的サイト）
- ✅ アクセシビリティ対応
- ✅ JavaScriptフレームワーク不要

## 📁 ファイル構成

```
├── index.html       # メインのHTMLファイル
├── styles.css       # レスポンシブスタイルシート
├── script.js        # JavaScriptロジック
└── README.md        # このファイル
```

## 🚀 使い始める

### 1. ファイルを用意する
- `index.html`
- `styles.css`
- `script.js`

これらのファイルを同じディレクトリに配置してください。

### 2. ブラウザで開く
`index.html`をブラウザで開くだけで、すぐに使用できます。

```
ファイル → 開く → index.html を選択
```

または、ローカルサーバーで実行：

```bash
# Pythonがある場合
python -m http.server 8000

# Node.jsがある場合
npx http-server
```

その後、ブラウザで `http://localhost:8000` にアクセスしてください。

## 📱 レスポンシブブレークポイント

- **デスクトップ**: 1200px以上
- **タブレット**: 768px以上 1200px未満
- **スマートフォン**: 768px未満

## 🎨 カスタマイズ方法

### カテゴリーの変更

`script.js`のカテゴリーラベルを編集：

```javascript
const labels = {
    'your-category': '🎯 あなたのカテゴリー',
    // ...
};
```

### トピックデータの追加

`script.js`の`topicsData`配列に新しいトピックを追加：

```javascript
{
    id: 9,
    title: "新しいトピックのタイトル",
    description: "説明文をここに入力",
    category: "tech",
    date: "2026-03-29",
    image: "https://example.com/image.jpg"
}
```

### 色スキームの変更

`styles.css`の色変数をカスタマイズ：

```css
/* グラデーション色の変更 */
.header {
    background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
    /* 新しい色に変更 */
}

/* アクセント色の変更 */
.filter-btn.active {
    background-color: #667eea; /* 新しい色に変更 */
}
```
